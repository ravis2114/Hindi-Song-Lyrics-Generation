#Hindi-Song-Lyrics-Generation
----
data is scrapped from internet

#Model
----
LSTM (Stacked)

##To Run:
----
```
python clean_data.py
```
To Train and Create Model (I'd advise running it in colab TPU if you don't have super fast GPU):
```
python lyrics_generator.py
```
